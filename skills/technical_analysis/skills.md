# Technical Analysis Skill
Calculates common technical indicators for stock analysis.

## Tools
- `calc_rsi(symbol, period=14)`: Calculate Relative Strength Index.
- `calc_macd(symbol)`: Calculate MACD (Moving Average Convergence Divergence).
- `calc_bollinger(symbol)`: Calculate Bollinger Bands.
- `scan_signals(symbol)`: Consolidated technical analysis report.
